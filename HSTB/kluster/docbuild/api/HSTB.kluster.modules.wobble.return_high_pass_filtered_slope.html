return_high_pass_filtered_slope¶

HSTB.kluster.modules.wobble.return_high_pass_filtered_slope(y, z, max_period, numtaps=101)¶

Perform linear regression on acrosstrack/depth offsets to get ping slope. High pass filter the slope to retain only the signal related to attitude.

Parameters

  • y (array) – numpy array (ping, beam) for alongtrack offset rel RP

  • z (array) – numpy array (ping, beam) for depth rel wline

  • max_period (float) – float, max period of the attitude arrays (roll, pitch, heave)

  • numtaps (int) – filter length, must be odd

Returns

  • np.array – numpy array (ping) high pass filtered slope for each ping

  • np.array – numpy array (ping) percent deviation of the model

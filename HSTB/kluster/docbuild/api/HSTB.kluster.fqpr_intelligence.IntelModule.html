IntelModule¶

class HSTB.kluster.fqpr_intelligence.IntelModule(**kwargs)¶

Bases: HSTB.kluster.logging_conf.LoggerClass

Base module for the intelligence modules. Intelligence modules are classes that contain a specific kind of file, and have methods that are useful for that file type to determine which files go together and which files should be processed in to which date/time/sonarmodel container.

Contains the attribution that all other extended modules share as well as the basic adding/removing functionality.

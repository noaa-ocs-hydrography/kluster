compute_geo_beam_pointing_angle¶

HSTB.kluster.modules.beampointingvector.compute_geo_beam_pointing_angle(bv_geo, rx_angle)¶

Build new beam pointing angle (rel to the vertical) and with the correct sign (+ to starboard) in the geographic ref frame.

Parameters

  • bv_geo (DataArray) – beam vectors in geographic ref frame, of shape (time, beam, bv_xyz), see build_geographic_beam_vectors

  • rx_angle (DataArray) – receiver beam pointing angle for each ping time

Returns

2 dim (time, beam) values for beampointingangle at each beam

Return type

xr.DataArray

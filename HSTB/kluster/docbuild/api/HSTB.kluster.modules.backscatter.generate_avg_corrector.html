generate_avg_corrector¶

HSTB.kluster.modules.backscatter.generate_avg_corrector(corrected_bscatter, beam_angles_degrees, bin_size_degree=1.0, reference_angle=45)¶

Compute a new angle varying gain corrector for the provided processed backscatter dataset. This corrector is then used during backscatter surface generation to remove the angle dependance in the processed backscatter.

Parameters

  • corrected_bscatter (DataArray) – processed backscatter returned from one of the Bscatter classes below, 2d array

  • beam_angles_degrees (DataArray) – corrected beam angles for the beams in degrees, 2d array

  • bin_size_degree (float) – size of the bin used to generate the corrector, in degrees

  • reference_angle (float) – angle used to determine the reference backscatter level

Returns

dictionary of {angles (degrees): avg correctors (dB)}

Return type

dict

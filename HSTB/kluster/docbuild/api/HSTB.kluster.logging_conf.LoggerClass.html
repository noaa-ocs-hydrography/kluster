LoggerClass¶

class HSTB.kluster.logging_conf.LoggerClass(silent=False, logger=None)¶

Bases: object

Basic class for logging. Include a logging.logger instance to use that, or set silent to true to disable print messages entirely. Use of Logger will trump silent.

Methods Summary

print_msg(msg[, loglvl])

Either print to console, print using logger, or do not print at all, if self.silent = True
